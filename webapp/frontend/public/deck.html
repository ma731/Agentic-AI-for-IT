02 · The solution

Six agents that decide, not just dashboards that show

  • Orchestrator (LLM) routes 5 autonomous ReAct specialists.
  • They converse through one shared transcript; agents can ask each other follow-ups.
  • Guided autonomy: a policy guarantees coverage + termination; the LLM picks order and each agent chooses its own tools.
  • Built on LangGraph (StateGraph + interrupt + checkpointer).
02 · Architecture
03 · The demo — the Friday Cascade

Perceive → reason → act → learn

  • Perceive: triage 22k alerts → CNC-07-LEI, RUL 52–76h.
  • Reason: recall precedent (INC-0288), source parts by ROI, reroute, gate vs OSHA.
  • Act: one costed plan; spend over €500 pauses for a human (code-enforced).
  • Learn: each closed run is logged to case memory and reconciled against the outcome.

Three paths: Cascade (human gate) · Edge (autonomous under €500) · Escalation (abstains on thin data).

05 · Next steps

From MVP to plant-wide

  • Wire real telemetry + CMMS/ERP tools (today's tools are scenario-faithful stubs).
  • Train the RUL estimator on real failure history (today: transparent heuristic + pattern-match).
  • Close the learning loop in production: auto-reconcile outcomes, tune confidence from human decisions.
  • Roll out across the fleet (CNC-03/05/08, robot cells) and other plants.

The architecture is the contribution. The model, the tools, and the data sources are swappable.
